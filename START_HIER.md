# START HIER - CC Alpha 0.1 Android

Dit is de nieuwe officiële startbasis voor CC als native Android-app.

## Wat je opent
Open in Android Studio de map:

`CC_Alpha_0_1_Android`

Niet één los bestand. Een Android-app is een projectmap.

## Eerste test
1. Open Android Studio op je laptop.
2. Kies `Open`.
3. Selecteer de map `CC_Alpha_0_1_Android`.
4. Wacht tot Gradle klaar is.
5. Klik op `Run` of bouw een APK via:
   `Build > Build Bundle(s) / APK(s) > Build APK(s)`.

## Wat Alpha 0.1 bevat
- Native Android-project in Kotlin + Jetpack Compose.
- Landscape-first voor Samsung Galaxy Tab S10 Lite.
- CC-logo uit het logodocument als visuele basis.
- Focus als startscherm.
- Schrijven als primaire actie.
- S Pen-schrijfcanvas als eerste basis.
- Voorstel van CC.
- Master Card met notities, levensloop en acties.

## Wat Alpha 0.1 nog niet bevat
- Geen echte AI.
- Geen cloud.
- Geen database.
- Geen Play Store-publicatie.

## Productregel
CC is een plannings- en actiemanager. Schrijven is de snelste invoermethode.
