# CC Alpha 0.1 - Native Android

CC is een Android Command Center dat gedachten omzet in overzicht.

Het is geen notitie-app en geen standaard taken-app. Het is een plannings- en actiemanager waarbij schrijven met de S Pen de snelste invoermethode is.

## Alpha 0.1
Deze versie bevat de eerste native Android-basis:

- Kotlin + Jetpack Compose
- Landscape-first tablet-layout
- CC-branding met logo uit het logodocument
- Focus-scherm
- Schrijven met eerste pen-canvas
- Voorstel van CC
- Master Card met notities, levensloop en acties

## Openen
Open de projectmap in Android Studio.

## APK bouwen
Android Studio: `Build > Build Bundle(s) / APK(s) > Build APK(s)`.

De APK komt daarna in:

`app/build/outputs/apk/debug/app-debug.apk`
